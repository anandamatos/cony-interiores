# 📋 Template de Report de Cerimônias Ágeis (Assíncrono - Discord)

Este template foi criado para ser usado em um canal do Discord, permitindo que todos os membros do time contribuam e acompanhem o progresso de forma assíncrona.

---

## 1. Sprint Planning (Início da Sprint)

**Quando:** Primeiro dia da Sprint  
**Canal:** `#sprint-planning`

### Report Template:

```
🚀 **SPRINT PLANNING - [MVP X - Sprint Y: Nome da Sprint]**
📅 **Data:** [DD/MM/YYYY]
👥 **Participantes:** @menção_dos_participantes

---

### 🎯 Sprint Goal
[Descreva o objetivo principal da Sprint, de forma clara e concisa]

---

### 📋 Backlog da Sprint

| Story ID | Título | Squad | Estimativa (SP) | Responsável |
|----------|--------|-------|-----------------|-------------|
| STORY-XX-XXX-001 | [Título da Story] | [Squad] | [N] | @menção |
| STORY-XX-XXX-002 | [Título da Story] | [Squad] | [N] | @menção |

**Total de Story Points:** [N]

---

### 🔍 Discovery & Mensuração (Liderança)

| Tarefa | Responsável | Status |
|--------|-------------|--------|
| DISCOVERY-XX-XXX-001: [Descrição] | @menção | 🔄 Em andamento |
| MEASUREMENT-XX-XXX-001: [Descrição] | @menção | ⏳ Pendente |

---

### 📌 Riscos e Dependências
- [ ] [Descreva riscos ou dependências identificados]

---

### ✅ Compromissos
- [ ] [Compromissos assumidos pelo time para esta Sprint]

---

🔗 **Links Úteis:**
- GitHub Project: [link]
- Documentação: [link]

**Próximos Passos:**
- [ ] Iniciar desenvolvimento das stories
- [ ] Dailies assíncronas no canal `#daily`

```

## 2. Daily (Atualização Diária)

**Quando:** Todos os dias (até às 10h)  
**Canal:** `#daily`

### Report Template:

```
📢 **DAILY - [DD/MM/YYYY]**
👤 **Membro:** @seu_nome

---

### ✅ O que fiz ontem?
- [ ] [Atividade concluída 1]
- [ ] [Atividade concluída 2]

### 🚧 O que vou fazer hoje?
- [ ] [Atividade planejada 1]
- [ ] [Atividade planejada 2]

### ⚠️ Bloqueios?
- [ ] [Descreva bloqueios, se houver] / Nenhum bloqueio

### 📊 Estou no caminho para cumprir meu compromisso da Sprint?
- [ ] Sim / Não (se não, explique)
```

### Exemplo de Uso:

```
📢 **DAILY - 28/06/2026**
👤 **Membro:** @anandamatos

---

### ✅ O que fiz ontem?
- [ ] Finalizei o protótipo do dashboard financeiro no Figma
- [ ] Validei as cores com a Cony Interiores
- [ ] Criei as issues de Discovery para o MVP 2

### 🚧 O que vou fazer hoje?
- [ ] Revisar os PRs do backend
- [ ] Criar os cards de Measurement para o EPIC-M2-UX-001
- [ ] Atualizar o README com as novas decisões

### ⚠️ Bloqueios?
- [ ] Aguardando feedback da Karine sobre os KPIs financeiros

### 📊 Estou no caminho para cumprir meu compromisso da Sprint?
- Sim, estou dentro do planejado.
```
