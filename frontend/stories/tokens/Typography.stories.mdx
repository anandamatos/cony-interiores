# 🔤 Sistema Tipográfico

A tipografia da Cony Interiores combina uma fonte moderna e legível com uma fonte serifada para toques de elegância.

## Famílias de Fontes

| Família | Fonte | Uso |
|---------|-------|-----|
| **Primária** | Inter | Textos principais, títulos, corpo |
| **Secundária** | Playfair Display | Destaques, citações, elementos decorativos |

## Escala Tipográfica

| Tamanho | Rem | Pixels | Uso |
|---------|-----|--------|-----|
| xs | 0.75rem | 12px | Legendas, tags |
| sm | 0.875rem | 14px | Textos secundários |
| base | 1rem | 16px | Corpo de texto |
| lg | 1.125rem | 18px | Destaques |
| xl | 1.25rem | 20px | Subtítulos |
| 2xl | 1.5rem | 24px | Títulos de seção |
| 3xl | 1.875rem | 30px | Títulos principais |
| 4xl | 2.25rem | 36px | Títulos grandes |
| 5xl | 3rem | 48px | Títulos hero |

## Pesos de Fonte

| Peso | Valor | Uso |
|------|-------|-----|
| Thin | 100 | Uso limitado |
| Light | 300 | Textos leves |
| Normal | 400 | Corpo de texto |
| Medium | 500 | Destaques |
| Semibold | 600 | Subtítulos |
| Bold | 700 | Títulos |
| Extrabold | 800 | Títulos principais |
