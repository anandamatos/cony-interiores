# 📏 Sistema de Espaçamento

O sistema de espaçamento da Cony Interiores é baseado em uma escala de 4px, garantindo consistência visual em toda a interface.

## Escala de Espaçamento

| Token | Valor | Uso |
|-------|-------|-----|
| 0 | 0px | Sem espaçamento |
| 1 | 4px | Espaçamento mínimo |
| 2 | 8px | Espaçamento entre elementos próximos |
| 3 | 12px | Padding interno pequeno |
| 4 | 16px | Padding padrão |
| 5 | 20px | Espaçamento entre seções |
| 6 | 24px | Margem entre blocos |
| 8 | 32px | Separação de seções |
| 10 | 40px | Espaçamento de página |
| 12 | 48px | Seções grandes |
| 16 | 64px | Espaçamento hero |
| 20 | 80px | Seções principais |
| 24 | 96px | Espaçamento máximo |
